# Cheat Sheet

## Overview of all components

| Component       | Syntax                                                                                | Markdown Equivalent       | Props                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| --------------- | ------------------------------------------------------------------------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Link            | `<Link />`                                                                            | `[Text](https://link)`    | `to`: the URL  <br />`icon`: an icon from DAI-ui <br />                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Button          | `<Button />`                                                                          | None                      | `to`: the URL <br /> `variant`: A property from dai-ui <br /> `secondary`: use secondary styles <br /> `outline`: Use outline styles. **Can be combined with secondary to show secondary outline styles** <br /> `disabled`: Disable button interaction <br /> `inline`: Force the button to be 'display: inline-block' <br /> **block by default** <br /> `sx`: SX style props                                                                                                                   |
| Inline Code     | \`\`\`js:title=example.js                                                             | \`\`\`                    | Just remember that you can add titles and language highlighting!                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| CTA             | `<CTA />`                                                                             | `> `                      | None                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Callout         | `<Callout />`                                                                         | None                      | `secondary`: use secondary styles, **defaults to primary** <br /> `icon`: specify the icon to appear with text                                                                                                                                                                                                                                                                                                                                                                                    |
| Status Banner   | `<StatusBanner />`                                                                    | None                      | `warning`: orange <br /> `error`: red <br /> `notice`: yellow <br /> `color`: custom color i.e. #fff000 or "red" <br /> `sticky`: make the element use it's sticky version                                                                                                                                                                                                                                                                                                                        |
| Image           | `<Image />`                                                                           | `![Alt text](image link)` | None - use Aligner for styling                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Aligner         | `<Aligner />`                                                                         | None                      | `center`: Center the image on the X and Y <br /> `right`: Align the image to the right <br /> `bottom`: Align the image to the bottom <br /> `yCenter`: Center the image on the Y axis <br /> `xCenter`: Center the image on the X axis                                                                                                                                                                                                                                                           |
| Inline Video    | `youtube: mzDVaKRApcg` or <br /> `video: https://www.youtube.com/watch?v=mzDVaKRApcg` | None                      | Supports YouTube (as shown), Vimeo, videoPress.                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Checklist       | `<Checklist />`                                                                       | None                      | No props - just follow the Child Rule and you can put anything in here!                                                                                                                                                                                                                                                                                                                                                                                                                           |
| List            | `<List />`                                                                            | `-`  or `*`               | No props, but check out the [Intermediate tutorial](./intermediate) to see the power and flexibility of this component.                                                                                                                                                                                                                                                                                                                                                                           |
| Accordion       | `<Accordion />`                                                                       | None                      | `openIcon`: defaults to `+`, can be anything from DAI-ui though <br /> `closeIcon`: defaults to `-` <br /> `defaultOpen`: control whether the menu is open, or closed (default)                                                                                                                                                                                                                                                                                                                   |
| Process Stepper | `<Process />`                                                                         | None                      | No props, but check out the [Intermediate tutorial](./intermediate) to see the power and flexibility of this component.                                                                                                                                                                                                                                                                                                                                                                           |
| Box             | `<Box />`                                                                             | None                      | No props, but this is how we bundle together lots of children (i.e. new lines in mdx) and pass them as one thing to React components. Works like a `<div>` in html                                                                                                                                                                                                                                                                                                                                |
| Tout            | `<Tout />`                                                                            | None                      | `alt`: applies yellow alt color <br /> `color`: specify custom css colors/hex codes for the background. If a custom color prop is provided the copy will change to check against HexLuminance rules. <br /> `stroke`: removes background and applies a colored border of (default, alt, or color) <br /> `fade`: fades the background from the specified color (or default/alt variants) to transparent. <br /> `image`: applies 1 of 2 static images depending on the existance of the alt prop. |
| InfoBlock       | `<InfoBlock />`                                                                       | None                      | No props, but three kinds. <br /> 1. Image on left (put `<Image />` as first child) <br /> 2. Image on right (put `<Image />` as last child) <br /> 3. No image priority (`<Image />` neither first nor last). See our [Advanced tutorial](./advanced) for more.                                                                                                                                                                                                                                  |
